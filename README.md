# Cube3D v3.2.6

## Как использовать

1. Установи зависимости:
pip install ursina pygame

2. Запусти:
python launcher.py

3. Выбери 2D или 3D режим

## Структура

- `launcher.py` — лаунчер
- `main.py` — логика выбора режима
- `Scripts/world_3d.py` — сцена Ursina
- `Scripts/world_2d.py` — сцена Pygame

## Пример в 3D

- Игрок — куб
- Камера — от третьего лица
- Управление — W A S D

## Пример в 2D

- Игрок — квадрат
- Управление — W A S D
